JOKENPO!

Projeto feito em razão do Trabalho PjBL RA1 - Jokenpô, da matéria Raciocínio Algorítimico de Ciência da Computação.

O objetivo do trabalho é criar um programa em python que simule uma partida de Jokenpo, em três modalidades diferentes, humano contra humano, humano contra maquina e maquina contra maquina.
O programa acontece inteiramente ao terminal, codigo formado com diversos IF's e WHILE's e com uso de 3 bibliotecas: 
'OS', usado para limpar o terminal, independendo do sistema operacional,
'GETPASS', usado para esconder o input digitado do usuario, util para que os jogadores não vejam a jogado um do outro.
'RAMDOM', usado para aleatorizar a escolha de jogada da maquina.
'TIME', usado para dar um suspense ao jogo, melhorando a experiencia geral.


O progama inicia, mostrando uma arte ANCII da palavra jokenpo e explicando as regras, o jogador deve escolher a modalidade desejada inserindo os numeros referentes as opções e apertando ENTER.

Ao escolher a modalidade humano contra humano, o jogador 1 deve fazer sua jogada escolhendo a mão desejada, note-se que o input não é mostrado devido a biblioteca 'getpass'.
Logo após o jogador 2 deve escolher sua mão. É exibido 'JO' 'KEN' 'PO' e ao final da rodada é exibido o jogador vencedor e o placar atual, é adicionado 1 ponto ao placar do jogador vencedor,
note-se que ao final também é exibido as mãos dos jogadores por meio dos emojis. 🪨 = Pedra | 📜 = Papel | ✂️ = Tesoura
É dado a opção de continuar com a partida, os pontos são mantidos e a rodada é reiniciada, ao escolher sair, os pontos são zerados e volta a tela inicial.
